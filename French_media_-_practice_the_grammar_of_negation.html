Exercices de Négation

Les médias français : gagnants et perdants économiques

Instructions : Transformez ou complétez les phrases pour pratiquer les différentes formes de négation en français. Tapez votre réponse puis appuyez sur Entrée pour vérifier chaque phrase individuellement.

📚 Rappel : Quelques formes de négation utiles

  • ne... pas = négation simple
  • ne... plus = ne... pas + plus (plus maintenant)
  • ne... jamais = à aucun moment
  • ne... rien = aucune chose
  • ne... personne = aucun individu
  • ne... aucun(e) = pas un seul / pas une seule
  • ne... ni... ni... = ni l’un ni l’autre
  • ne... que = seulement (restriction)
  • ne... guère = très peu / presque pas (registre plutôt soutenu)

Section A : Négation de phrases affirmatives

Tâche : Mettez ces phrases à la forme négative la plus logique (ne... pas, ne... plus, ne... guère, etc.).

1. Les journaux papier génèrent encore beaucoup de revenus publicitaires.
2. La télévision domine totalement le marché publicitaire français.
3. Les médias traditionnels attirent les jeunes lecteurs.
4. La presse écrite peut survivre grâce aux revenus publicitaires.
5. Les plateformes numériques sont une menace pour les journaux.

Section B : Ne... jamais / Ne... rien / Ne... guère

Tâche : Complétez avec une négation appropriée (ne... jamais, ne... rien, ne... guère, etc.).

6. Les jeunes ________ lisent ________ les journaux papier.
7. Les plateformes numériques ________ paient ________ aux créateurs de contenu original.
8. La radio ________ fait ________ pour attirer les publicitaires.
9. Les médias traditionnels ________ investissent ________ dans le numérique.

Section C : Ne... personne / Ne... aucun(e)

Tâche : Transformez avec la négation demandée.

10. Quelqu'un fait confiance aux algorithmes des plateformes. (utilisez une négation avec « personne »)
11. La presse a des solutions pour résoudre sa crise économique. (utilisez une négation avec « aucun(e) »)
12. Quelqu'un peut arrêter la domination d'Internet. (utilisez une négation avec « personne »)
13. Les journaux ont un avenir prometteur. (utilisez une négation avec « aucun »)

Section D : Ne... ni... ni...

Tâche : Combinez les phrases avec ne... ni... ni... ou une structure équivalente.

14. La presse papier n'a pas d'argent. La presse papier n'a pas de lecteurs.
15. Les médias traditionnels ne comprennent pas les jeunes. Les médias traditionnels ne les intéressent pas.
16. La télévision n'attire pas les annonceurs. La radio n'attire pas les annonceurs.

Section E : Ne... que (restriction)

Tâche : Reformulez avec ne... que (idée de « seulement »).

17. Internet capte seulement 70 % des revenus publicitaires.
18. La radio maintient seulement 700 millions d'euros de recettes.
19. Les jeunes font seulement confiance aux influenceurs.
20. Les plateformes pensent seulement aux profits.

Section F : Négations multiples (avancé)

Tâche : Traduisez ou complétez en combinant plusieurs négations si nécessaire.

21. Traduisez : « The press no longer attracts anyone. »
22. Traduisez : « Nobody ever reads traditional newspapers anymore. »
23. Traduisez : « Internet platforms never give anything to journalists. »
24. Complétez : « Sans publicité, les journaux ________ peuvent ________ survivre. »
